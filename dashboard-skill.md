# 📊 국장 데일리 대시보드 스킬 파일

## 기본 원칙
- 검색으로 확인된 데이터만 사용 (추정값 절대 금지)
- 확인 못한 값은 표시하지 않거나 "장중 변동" 등으로 표기
- 날짜 표기는 항상 한글로 (3월 23일, 4월 등)
- 모바일 최적화 HTML로 제작 (max-width: 480px)
- 카카오톡 호환: 구글 폰트 사용 금지, 시스템 폰트 사용

## 데이터 수집 순서 (반드시 검색 확인 후 입력)

### 1. 국내 지수
- [ ] 코스피 종가, 등락폭, 등락률
- [ ] 코스닥 종가, 등락폭, 등락률
- [ ] 오늘 장중 특이사항 (사이드카, 서킷브레이커 등)

### 2. 수급 (코스피 기준)
- [ ] 외국인 순매수/순매도 금액
- [ ] 기관 순매수/순매도 금액
- [ ] 개인 순매수/순매도 금액
- [ ] 외국인 선물 포지션 방향

### 3. 매크로 지표 (전일 대비 변화 + 한줄 이유 필수)
- [ ] USD/KRW 환율 · 전일比 변화 · 이유
- [ ] 브렌트유 · 전일比 변화 · 이유

- [ ] 금 선물 · 전일比 변화 · 이유
- [ ] 달러 인덱스 · 전일比 변화 · 이유

### 4. 해외 지수
- [ ] S&P 500 종가, 등락률
- [ ] 나스닥 100 종가, 등락률
- [ ] 다우존스 종가, 등락률

### 5. 당일 특징주 (검색 확인된 종목만)
- [ ] 상한가 또는 급등 종목
- [ ] 급락 주의 종목
- [ ] 핫 테마

### 6. 뉴스 (기사 URL 반드시 포함)
- [ ] 국내 주요 뉴스 5~7건
- [ ] 해외 주요 뉴스 4~5건
- [ ] 부동산 뉴스 3~4건

## 탭 구조

```
📈 국내 | 🇺🇸 해외 | 🏠 부동산 | 🔥 핫이슈
```

### 각 탭 필수 구성요소

**📈 국내**
1. 오늘 시황 한줄 요약 배너 (긴급 이슈 시 빨간색)
2. 코스피·코스닥 지수 카드
3. 매크로 지표 그리드 (전일比 % + 한줄 이유)
4. 수급 분석 바차트 (외국인/기관/개인 + 프로그램매매 + 선물)
5. 섹터 동향 바차트
6. 당일 특징주·테마
7. 시총 상위 종목 테이블
8. 국내 뉴스 (링크 포함)
9. 향후 일정 캘린더

**🇺🇸 해외**
1. 미국 시황 한줄 요약 배너
2. 미국 3대 지수 + 닛케이 카드
3. 글로벌 매크로 지표 (전일比 % + 한줄 이유)
4. 해외 섹터 동향
5. 해외 주요 종목 테이블
6. 해외 뉴스 (링크 포함)
7. 실적 발표 일정

**🏠 부동산**
1. 부동산 핵심 요약 배너
2. 핵심 지표 그리드
3. 지역별 바차트
4. 부동산 뉴스 (링크 포함)
5. 향후 일정

**🔥 핫이슈**
1. 오늘 핵심 이슈 요약 배너
2. 핫 키워드 태그
3. 오늘 일정 타임라인
4. 핫이슈 뉴스 (링크 포함)
5. 향후 일정

## 📍 송파구 단지 시세 트래커 (고정 섹션)

부동산 탭에 항상 포함되는 고정 단지 시세 카드.
매번 대시보드 만들 때 아래 검색으로 최신 실거래가 업데이트할 것.

---

- **위치**: 서울 송파구 거여동 · 5호선 거여역 도보 6분
- **기본 정보**: 546세대 · 1997년 입주 · 6개동 · 투기지역
- **평형**: 17평(49㎡) · 21평(59㎡) · 26평(72㎡)

**검색 키워드**
```
거여4단지 실거래가 [연월]
거여4단지 매매 시세 최근
```

## 단지 시세 카드 업데이트 방법

매번 대시보드 제작 시 아래 항목 검색 후 업데이트:

```
1. 거여4단지 실거래가 최근 → 평형별 최신 거래 반영
2. 문정시영 실거래가 최근 → 평형별 최신 거래 반영
3. 문정시영 리모델링 현황 → 진행 단계 변경 여부 확인
```

카드 레이아웃:
- 상단: 단지명 + 기본정보 + 상태 뱃지
- 중단: 평형별 시세 그리드
- 하단: 최근 실거래 타임라인 3건
- 하단: 리모델링 진행 단계 (문정시영만)
- 최하단: 투자 포인트 한줄 요약
